# Мини-приложение MAX

Frontend-приложение на React для работы с визитами на склад. Backend и `Process.aspx` не используются напрямую: все данные приходят из mock API слоя `src/api/processApi.js`.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Проверка консоли браузера

```bash
npm run check:console
```

## Маршруты

- `/` — актуальный визит, поддерживает `?tvsid=123`
- `/history` — история визитов
- `/visit/:id` — выбранный визит
- `/change-time` — экран-заглушка переноса времени

Mock API сделан отдельным модулем, чтобы позже заменить функции на реальные HTTP-запросы без переписывания UI.
